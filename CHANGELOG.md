# Changelog

All notable changes to the **State of Command Editor** are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [v1.0] - 4/20/2026

First official version of the State of Command Editor.

### Added a City

- Added a discord server member's city (won't specify what city that is for obvious reasons).

### Editor

- Reference images lock, delete and hide controls are back.
- Map should always load on launch (some users were reporting that it doesn't load provinces until they swap between nation and province in settings, We believe the cause of the issue has been fixed).
- You can select nations by click on nation mode now.
- Province creation visuals are reinstated.
- Unit's composition is loaded properly.
- World texture is not clipped to mesh on purpose (for a smooth experience). This is only for the editor. World texture is still clipped properly in the main game (and runs smoothly there).
