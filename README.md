## Download

The latest build is always available from [releases](https://github.com/YoungLionIV/State-of-Command-Editor/releases/latest)

## Installation

1. Download the latest version's 7z file from [releases](https://github.com/YoungLionIV/State-of-Command-Editor/releases)
2. Extract the 7z file with [7-Zip](https://www.7-zip.org/) (or any other compatible tool) into a folder
3. Run the editor executable from the extracted folder
4. Enjoy
